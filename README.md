# Atharva Jha - Personal Portfolio

A modern, interactive, and fully responsive personal portfolio website showcasing my skills, projects, coding profiles, and achievements.

### 🌐 Live Demo
You can view the live deployed site here: **[https://atharvajha21.github.io/my_portfolio/](https://atharvajha21.github.io/my_portfolio/)**

### ✨ Highlights & Features
- Beautiful UI with smooth glassmorphism effects and modern styling.
- Interactive particle canvas background.
- Sections dedicated to my Tech Stack, Projects (including Nova AI and Smart City Dashboard), and competitive programming profiles.
- Integrated contact form.

### 🛠️ Built With
- HTML5
- CSS3 (Custom styling, animations, responsive design)
- JavaScript (Vanilla JS for interactive elements)
- 🚀 **Made using Antigravity & Vibe Coding** ✨
